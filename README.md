## recoil-to-do
recoil을 이용해 만든 to-do-list입니다. 

## 배포 링크
https://fe012.github.io/recoil-to-do

## 기술 스택
React, TypeScript, Recoil, Styled-components

## 개발 환경 세팅
```bash
npm install
npm start
```
